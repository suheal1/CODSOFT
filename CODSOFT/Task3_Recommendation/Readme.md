# Movie Recommendation System

This project implements a simple content-based recommendation system using cosine similarity.

## Features
- Text vectorization using CountVectorizer
- Cosine similarity computation
- Top 3 similar movie recommendations
- Case-insensitive search
- Interactive command-line interface

## Algorithm Used
Content-Based Filtering:
Movies are compared based on genre similarity using vector space modeling.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Cosine Similarity

## How to Run
python recommendation.py
